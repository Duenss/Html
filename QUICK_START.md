# 🚀 Inicio Rápido - 60 Segundos

## 1️⃣ Abre la Aplicación

### Opción A: Localmente
- Descarga los archivos
- Abre `index.html` en tu navegador
- ¡Listo!

### Opción B: Netlify (Recomendado)
- Arrastra la carpeta a [netlify.com](https://netlify.com)
- Tu sitio estará en vivo en segundos
- Compartible por URL

## 2️⃣ Crea tu Primer Embed

### Paso 1: Edita el Título
1. Haz clic en el botón **"Título"** en el panel izquierdo
2. Se abrirá un modal elegante
3. Escribe tu título (máx. 256 caracteres)
4. Haz clic en **"Guardar"**

### Paso 2: Edita la Descripción
1. Haz clic en **"Descripción"** en el panel izquierdo
2. Se abrirá otro modal
3. Escribe tu descripción (máx. 4096 caracteres)
4. Haz clic en **"Guardar"**

### Paso 3: Cambia el Color
1. Haz clic en **"Color"** en el panel izquierdo
2. Se abrirá el selector de color
3. Elige tu color preferido o pega un hex (ej: #8A2BE2)
4. Haz clic en **"Guardar"**

### Paso 4: Ver la Previsualización
- Mientras editas, verás los cambios en **tiempo real** en el centro
- El embed se actualiza instantáneamente

## 3️⃣ Guarda tu Embed

Una vez terminado:

1. Haz clic en **"💾 Guardar Embed"** en el panel derecho
2. Verás una notificación confirmando el guardado
3. El embed se guardará en tu navegador
4. Puedes crear más embeds

## 4️⃣ Copia el Código

Cuando quieras usar el embed en tu bot:

1. Haz clic en **"📋 Copiar JSON"**
2. El código se copiará automáticamente
3. Pégalo en tu bot de Discord
4. ¡Listo!

## 5️⃣ Usa Plantillas (Opcional)

Si quieres empezar más rápido:

1. Haz clic en **"Plantillas"** en el header
2. Selecciona una:
   - 👋 Bienvenida
   - 📢 Anuncio
   - ❌ Error
   - ✅ Éxito
3. Se cargará automáticamente
4. Personaliza según necesites

## 📂 Panel Izquierdo - Elementos

Los 8 botones permiten editar cada campo:

| Botón | Campo | Límite |
|-------|-------|--------|
| 📝 | Título | 256 caracteres |
| 📄 | Descripción | 4096 caracteres |
| 🎨 | Color | Hex o picker |
| 🖼️ | Imagen | URL |
| 🖼️ | Thumbnail | URL |
| 👤 | Autor | 256 caracteres |
| 📌 | Pie de página | 2048 caracteres |
| 🔗 | URL | URL del embed |

## 👀 Panel Centro - Vista Previa

Ves exactamente cómo se verá en Discord:

- Bot avatar
- Nombre del bot
- Barra de color
- Título
- Descripción
- Imagen (si proporciona)
- Thumbnail (si proporciona)
- Información del autor
- Pie de página

**Se actualiza en tiempo real** mientras editas.

## 🎯 Panel Derecho - Acciones

Tres botones principales:

| Botón | Acción |
|-------|--------|
| 💾 Guardar Embed | Guarda el embed actual |
| 📋 Copiar JSON | Copia el código |
| 🔄 Limpiar | Limpia todos los campos |

## 🌙 Cambiar Tema

En el **header superior derecho**:

1. Haz clic en el botón de **tema** (sol/luna)
2. Cambia automáticamente
3. Tu preferencia se guarda

## 💡 Tips y Trucos

### Color Picker Rápido
- Haz clic en el campo de color para abrir el selector
- O pega directamente un código hex (sin # necesario)

### Contadores
- Verás en tiempo real cuántos caracteres has usado
- El límite máximo está claramente indicado

### Plantillas
- Las plantillas son puntos de partida
- Puedes editar cualquier campo después de cargar

### Guardar Múltiples
- Puedes crear y guardar embeds ilimitados
- Accede a todos desde "Mis Embeds Guardados"

### Modo Oscuro
- Activado por defecto
- Cambia automáticamente con tu preferencia del sistema
- La elección se guarda

## 🔄 Flujo Completo en 5 Pasos

```
1. Editar Título (Click → Modal → Guardar)
2. Editar Descripción (Click → Modal → Guardar)
3. Editar Color (Click → Picker → Guardar)
4. Ver en Centro (Actualización en vivo)
5. Guardar Embed (Un clic → Confirmado)
```

## 🆘 Solución Rápida de Problemas

| Problema | Solución |
|----------|----------|
| No se ve la previsualización | Recarga la página |
| El archivo no se carga | Asegúrate de tener todos los archivos: `index.html`, `styles.css`, `script.js` |
| Los datos se borran | Usa otro navegador sin incógnito |
| Colores extraños | Intenta actualizar (Ctrl+F5) |
